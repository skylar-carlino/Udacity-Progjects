Lesson Zero (Getting Started with HTML)

Important Concepts of Lesson Zero

How does the web actually work?

The web is a bunch of computers that talk to each other through programs called web browsers, like Chrome and Firefox. This communication is called HTTP. The web browser talks to the server, which is basically a computer that only stores files. The server finds the correct HTML document and sends it back to your browser. The computer interprets the page and displays it on your screen.

HTML

HTML stands for Hypertext Markup Language. It is the basis for everything on the web.

HTML is made of :
  • Text content - What you see
  • Mark up - What it looks like
  • References to other documents = eg. images and videos
  • Links to other pages
    •

    HTML Markups

    <name > content < / name >

    Opening tag - < name >

    Closing tag - < / name >

    Common Markups
  • <b>-bold
  • <em>-italics
  • <u>-underlined
  • <a>- called anchor, used for making links
  • <img>-images
  • br- line breaks. Like the enter key when typing a new line in text.
  • <p> - making text into paragraphs.
  • <xmp> - used to show code.
    •

    Inline Vs. Block

    Inline text is displayed without line breaks. Block tags have an invisible block around them, and normally start and end with a new line. *Credit toward the discussion forums for this definition.

    The following are inline elements. A, BR, EM, B, IMG, and SPAN.

    The following are block elements. P, DIV, and FORM.

    Example HTML Document

    <!Doctype HTML> - Doctype - What kind of HTML it is. <HTML> - surrounds the entire document. <head> - meta data like java and CSS. <title> - title of the document. </title> </head> <body> - contents of the document. </body> </html>

    What is a web page?

    A web page is a text document written in HTML, or Hyper Text Markup Language. Web browsers, like Firefox, read these documents and then interpret and display them.

    How does coding work?

    Programmers write text in a language that a computer can understand.

    Computers are stupid!

    Don't forget to close tags! Computers think literally, and won't catch mistakes.

    Programmers can't remember everything.

    Use google, the discussion boards, or ask for help if you are stuck. There are many resources out there. Use them!

    Basic HTML Vocab

  • Tag - Contained with angled brackets. < >

  • Opening Tag - <content>
  • Closing Tag - </content>
  • Void Tags - Tags that don't require closing tags, like <br> or <img>
  • Element - A set of opening and closing tags.

  • Attribute - a property of an HTML element.

    Many Languages

    HTML, CSS, and Java Script are each languages used in web design. Each has a different fuction.

    Jessica provided a really good example.

    "A web page is like a house.

    HTML is structure. Where are the walls placed, which room is the kichen, bathroom, etc.

    CSS is style. What color is the carpet and walls? What kind of decorations are there?

    Java Script is the interactive components, like the tv remote or garage door opener."

    DOM

    DOM's are document object models. They are a visual aid for programmers, helping to make sure the information is in the correct order.

    CSS

    CSS, or Cascading Style Sheets, are used to add style to your HTML document. Without CSS, a document is dull and boring. CSS makes documents look exciting and interesting.

    CSS adds color, font, borders, text allignments, and so much more.
